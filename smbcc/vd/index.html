『SMBC経営懇話会』とは・・・

三井住友銀行のお取引先の経営課題解決のための「経営情報の発信」
および「企業間の交流」を目的としたマネジメントクラブです。
会員企業さまの事業発展を願い、多彩なサービスをご用意しておりますので、
経営者・経営幹部から若手社員まで、皆さまでご活用ください。

三井住友銀行 Value Door IDをお持ちのお客さまがご利用いただけるサービス

三井住友銀行 Value Door ID をお持ちのお客さまは、「SMBC経営懇話会」会員向けサービスの一部を無料でご利用いただけます。

ネットプレス

税金や法律など経営に関するタイムリーで身近なトピックスを解説しています。
A4サイズ1～2枚程度にまとめていますので、ちょっとした空き時間にお読みいただけます。

ネットプレスについて

一口メモ

日常の光景や歴史のエピソード、季節の話題等、様々なトピックスを題材としたコラムです。
朝礼時の挨拶や経営のヒントとしてご利用いただけます。

実務シリーズ

経営実務に役立つ実践的ノウハウ満載の当社オリジナル書籍です。
社員教育や自己啓発教材、社内研修テキストとしてもご好評いただいております。

SMBC経営懇話会特別会員の皆さまへ毎月お届けしている会報誌です。
経営者インタビューや最新テクノロジーの紹介、人材育成をテーマとした連載など、中堅・中小企業の経営課題解決に役立つ情報をお届けいたします。

スタートアップ会員のご案内

起業して間もないスタートアップ・アーリーステージの企業さまに、創業期にありがちな課題解決に役立つ豊富なメニューを、安価で提供するマネジメントクラブです。
会員メニューの他にも、SMBCグループや多彩な外部パートナーとの連携により、様々なコンテンツやイベントをご用意しております。
